# ai-pgm

A kit of **20 Claude Code skills** for Program Managers and TPMs — full program-lifecycle
automation grounded in real program data from Slack, Smartsheet, Drive, Jira, Calendar, and Zoom.

**Live:** https://adrianhills.github.io/ai-pgm/

The skills cover the whole lifecycle — kickoff, capacity planning, workback scheduling, RAID,
premortems, escalations, status updates, decision records, retrospectives, and RCA — and chain
together so one skill's output is the next one's input. Each skill is built for reliability:
verify-coverage checks, freshness gates, provenance back to source, and an A/B eval harness so a
skill only ships if it's materially better than the base model.

This is a single-page, zero-dependency static site. The catalog is filterable by lifecycle phase
and by the TPM/PGM career competency each skill builds.

Built by [Adrian Hills](https://www.linkedin.com/in/hillsadrian/) — proof, not portfolio.
