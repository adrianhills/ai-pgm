How skills chain together

The leverage isn't any single skill — it's the chains, where one skill's output is the next one's input.

🔄 Weekly operating rhythm

tpm-rockstar → comms → status-update

The chief-of-staff brief surfaces what shifted and drafts replies; triage prioritizes the inbox sweep; status-update drafts the executive summary.

🚨 Risk escalation path

tpm-rockstar → raid → premortem → escalate → decision

Brief detects the signal, RAID captures it, premortem stress-tests failure modes, escalation flags leadership with options, decision documents the resolution with DACE roles.

🚀 New program launch

kickoff → stakeholder-map → workback → kickoff → comms

Brief defines scope and DACE; stakeholder map identifies who's in the room; workback computes the schedule from the launch date; kickoff agenda runs the room; comms drafts the announcement.

✅ Meeting to action

zoom-meeting-summary → raid → comms

The transcript becomes a structured summary with action items and a mini RAID; the RAID log absorbs the new items with owners; comms posts to the right channel.

🏁 Program close

retro → comms → roadmap-compare

Retro captures lessons; spotlight recognizes contributors; roadmap compare feeds the lessons into the next planning cycle.

🗓️ Quarterly planning

business-case → roadmap-compare → capacity-plan → kickoff → decision

Business case validates each bet; roadmap compare detects collisions; capacity plan surfaces SPOFs; kickoff briefs scope the approved bets; decision documents the portfolio call.

🔧 Incident to learning

rca → retro → premortem → raid

RCA structures the timeline and corrective actions; retro captures broader reflections; premortem applies the learnings to prevent recurrence; RAID absorbs the corrective actions with target dates.

Built for reliability

A kit you delegate real work to needs reliability primitives and an eval harness — not just good prompts.

🔍

Reliability primitives

Verify-coverage checks, freshness gates on stale data, and provenance back to Slack / Smartsheet / Drive / Jira / Calendar. Every claim traces to a source.

🧪

Eval harness

Every skill is A/B tested against the same request with no skill loaded. If it isn't materially better than the base model, it doesn't ship.

📈

Career competency mapping

Each skill maps to the TPM/PGM career competencies it builds, so delegated work also reinforces the competencies you're promoted on.

🧠

Program memory

A per-program Brief re-injected at run start, plus an append-only Decisions Ledger with mandatory evidence refs. Contradictions are surfaced, never silently resolved.
